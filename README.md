## Сборка проекта

### Локальная сборка через npm

Требования:

- node 20 и выше
- npm 10 и выше

Порядок запуска:

- ставим зависимости: `npm install`
- запускаем приложение: `npm run dev`
- переходим на http://127.0.0.1:5173

Запуск storybook:
- запускаем командой: `npm run storybook`
- переходим на http://127.0.0.1:6006
